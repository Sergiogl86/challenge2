Home ¿Quienes somos? Nuestro CV Contacto

Week 1 - Challenge 2


Este es un challenge para hacer por parejas.
El objetivo es crear con HTML (y nada de CSS) una web con cuatro páginas: Home, Quiénes somos, Nuestros CV y Contacto.
Todas las páginas deben tener la misma cabecera, con los siguientes elementos: - Nombre de la web - Menú de navegación con links para ir a las tres secciones.
La home puede llevar el contenido que queráis. La sección Quiénes somos debe contener dos fichas, una para cada miembro, y en cada ficha debe aparecer una foto, el nombre y alguna información.
La sección Nuestros CV debe contener vuestros dos CV. La sección Contacto debe llevar un formulario que pida Nombre, Email y Comentarios (el email tiene que ser obligatorio).
